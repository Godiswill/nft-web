# Nextjs 模板

## 技术栈

-   JS 框架：[Nextjs](https://nextjs.org/)
-   CSS：Sass + CSS-in-JS + [TailwindCSS](https://www.tailwindcss.cn/docs) (学会这个基本不需要前两者)

## 使用

开发

```
npm run dev

http://localhost:3000

```

build

```
npm run build
```

eslint

```
npm run lint
```

开发包分析

```
npm run analyze-dev
```

build 包分析

```
npm run analyze
```
